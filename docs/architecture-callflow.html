Primary flow — Add Place (UI → Process → Integration → Business → DB)
1
Frontend · AddPlaceForm.tsx
User selects country and types city name. Clicks "Add place."
useCreatePlace().mutate({ cityName, countryIsoAlpha2, isHome })
↓
2
Frontend · API Client
POST /api/places  ·  body: AddPlaceDto { cityName, countryIsoAlpha2, isHome }
↓
3
Functions · PlaceFunctions.CreatePlace
Deserializes body. Delegates to process. Maps exceptions to HTTP codes.
await IAddPlaceProcess.ExecuteAsync(dto, ct)
NotFoundException → 404  |  BusinessRuleException → 422
↓
4
Process · AddPlaceProcess.ExecuteAsync — orchestrates the full operation
CountryBusiness.GetByIsoAlpha2Async(countryIsoAlpha2)  → NotFoundException if not found
NominatimGeocodingService.GeocodeAsync(cityName, countryIsoAlpha2)  → step 5
validate: geocoded city name must contain words from input  → BusinessRuleException if mismatch
PlaceBusiness.CreateAsync(lon, lat, countryId, city, stateAbbr, isHome)  → step 6
if country not yet visited: CountryBusiness.SetVisitedAsync(countryId, true)  → step 7
↓
5
Integration · NominatimGeocodingService.GeocodeAsync — external HTTP call
Calls Nominatim OpenStreetMap API. Returns coordinates, canonical city name, state abbreviation.
GET nominatim.openstreetmap.org/search?q={city}&countrycodes={country}&addressdetails=1&limit=1
extracts: lat, lon, city (city → town → village → municipality → input fallback)
extracts state: ISO3166-2-lvl4 ?? state_code  → strips prefix (BR-PR → PR)
returns GeocodingResult or null
↓
6
Business · PlaceBusiness.CreateAsync
InsertAsync(new Place { Lon, Lat, CountryId, City, StateAbbr, IsHome })
→ PlaceDto
↓
7
Business · CountryBusiness.SetVisitedAsync (only if country was unvisited)
ExecuteUpdateAsync(c => c.Id == countryId, IsVisited = true)
↓
8
Frontend · React Query
201 Created → modal closes. Cache invalidated → places list and countries refresh automatically.
invalidateQueries(['places'])  ·  invalidateQueries(['countries'])
